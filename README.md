# nft_test

필요사항

1. node.js 

- version 16.13.2

2. mySql
- 소스에는 mysql기본 포트에 nft_test라는 db를 찾도록 되어 있다. (ID: root, PW: 1234)

  변경하려면 config폴더안의 config.json 파일 수정

3. alchemy  가입
- alchemy (https://www.alchemy.com/)에 가입 후 API url을 발급받아 /blockchain/nft_scripts.js 의 'API_URL'을 바꿔준다



설치

git clone 이후
npm install


실행

npm start
